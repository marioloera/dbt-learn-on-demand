Welcome to your new dbt project!

### Using the starter project

Try running the following commands:
- dbt run
- dbt run -m stg_payments
- dbt run -m stg_payments+
- dbt test
- dbt test --schema
- dbt test -m stg_customers
- dbt test --data
- dbt docs generate
- dbt source snapshot-freshness
- dbt deps
- dbt run --no-version-check
- dbt docs serve
- http://localhost:8080
